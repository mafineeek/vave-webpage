How to use variables?
{user.name} - Mentions user (eg. @vave)
{user.tag} - Member Tag (eg. vave#0000)
{user.id} - Member ID

{server.name} - Server name
{server.id} - Server ID
{server.avatarURL} - Server avatar (URL eg. https://cdn.vavebot.pl/favicon.png)



How to use inputs?
To mention channel type <#CHANNEL_ID>
To mention role type <@&ROLE_ID>
To mention other user type <@USER_ID>




How to use embed messages?
To provide an embed message type embed:{content} at start
To provide a message title start with title:
To provide a message description type description:{content}
To provide a message image type image:{content}
To provide a message footer type footer:{content} To provide a message embed color type color:{hex_color}
